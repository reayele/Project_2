## Project Description 

The Contact Manager allows the user to add, view, search, update, and delete contacts. The contact informations that are saved are put in a JSON file so the contacts are saved even after the program closes. 


# Features of the Program 

- Adding a contact 
- View all the contacts saved 
- Searching for a specific contact by name 
- Updating an existing contact 
- Deleting a contact 
- Saving Contacts into a JSON files
- Loading saved contacts when the program starts 
- Running automated test for the contact class
